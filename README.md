# ruxaen portfolio

A standalone personal portfolio page for Rooshan Samar.

The site opens with an interactive particle surface that transitions into a minimal portfolio interface with an about section, archive, social links, and contact email.

## Files

* `portfolio.html` - the complete portfolio page, including HTML, CSS, and JavaScript.
* `README.md` - project notes.

## Tech

* HTML
* CSS
* JavaScript

No build step is required. Open `portfolio.html` directly in a browser.
